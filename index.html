Proyecto Final

    Configurar la Red Anfitrion:
Para configurar la red anfitrion debemos acceder a la maquina VirtualBox, ir a "Herramientas globales" y una vez en ese apartado daremos click en "Crear" que se encuentra en la parte superior izquierda y una vez que hayamos hecho eso se haremos la tarjeta 2 y lo habilitamos. Una vez seleccionada la maquina virtual que creamos nos vamos a "Configuracion" y vamos al apartado de "Red", entonces clickeamos "Adaptador 2" y ahi Habilitamos adaptador de red dando click en el cuadrito, clickeamos "Conectado a red solo anfitrion" y cambiamos el nombre a "VirtualBox Host-only Ethernet Adapter #2", entonces el adaptador de red anfitrion listo para ser usado.

    Actualizacion de los paquetes:
Para actualizar los paquetes demebos insertar este codigo: "sudo apt update", al darle enter se buscaran las actualizaciones disponibles, introduciremos la contraseña y listo. Para instalar los paquetes tendremos que insertar el codigo siguiente: "sudo apt upgrade", daremos enter y cargaran los paquetes que podemos actualizar, nos pedira una confirmacion y daremos "s" y luego "Enter".

    Instalacion de paquetes de Ubuntu:
Ya que tengamos actualizados los paquetes de Ubuntu, procederemos a instalar los siguientes servicios PHP, MySQL, Apache con los siguientes comandos: sudo apt install php mysql-server apache2 php-mysql.

    Activacion de servicios:
Para Activar los servicios tenemos que poner estos comandos:

"sudo service apache 2 start"

"sudo service mysql start".

    Clonar el repositorio:
Si queremos clonar el repositorio nos movemos a la carpeta html con el comando "cd /var/www/html", ya que estemos ahi tendreemos que insertar el siguiente comando "sudo git clone https://github.com/dgeti-cetis108/Programacion-M4S2-2018.git", luego le vamos a cambiar el nombre con el comando: "sudo mv Programacion-M4S2-2018/library.com".

    Creacion de usuario MySQL:
Para crear un usuario usa el comando: "create user" 'ignacio' @'localhost' identified by '123 luego tememos que dar permisos de administrador con el comando: grant all on library.* to 'ignacio'@'localhost'.

    Modificar usuario y contraseña:
Para cambiar el usuario y la contraseña vamos a poner el siguiente comando: "cd library.com/classes" y "/library.com/classes$ sudo nano conexion.php" en esa ventana editaremos el usuario y contraseña que habiamos hecho.

    Configuracion del VirtualHost:

    Entramos a la carpeta /etc/apache2/sites-available , en la carpeta escribiremos "sudo nano library.com.conf" para poder modificarlo.

    "Dentro del archivo, escribe el siguiente codigo" :

      "VirtualHost*:80">

      "Server Name library.com"

      "Server Alias www.library.com"

      "Server Admin ignacio_zc@hotmail.com" (ejemplo)

      "DocumentRoot /var/log/library.com-error.log"

      CustomLog /var/log/library.com-acces.log. common"

      ServerName

      /VirtualHost>

      ADVERTENCIA: ANTES DE CADA VIRTUALHOST, LLEVA OTRA INVERSA A LA FINALIZADA.

    Revisa que tus carpetas estén en el siguiente orden /etc/apache2/sites-available/ escribiendo el comando ls .

    En la carpeta sites-available escribe: "sudo a2ensite library.com" (para habilitar el dominio)

    Reinicia el servicio de apache con el comando sudo service apache2 restart

    Entra a la carpeta /etc/hosts/ y agrega lo siguiente:

      127.0.0.1 library.com

      127.0.0.1 www.library.com

    Guarda los cambios con el comando "Ctrl+O"

    Una vez fuera de la maquina virtual en consola entra a la siguiente carpeta: /etc/hosts y agrega el ip de la red anfitrion + nombre del dominio. Por ejemplo:

      192.168.56.101 library.com

      192.168.56.101 www.library.com
